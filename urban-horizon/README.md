# Urban Horizon

**A soft light Omarchy theme** built from the blended palette of all wallpapers in `backgrounds/`.
The look is clean, calm, and grounded: muted stone + steel-blue accents on a bright neutral base.

## Palette (core)

- **Background:** `#ecebea`
- **Surface:** `#d6d8db`
- **Primary text:** `#2f3a42`
- **Secondary text / borders:** `#46545d` / `#9fa7ae`
- **Primary accent (muted bronze):** `#8a6f5c`
- **Secondary accent (steel blue):** `#5f7888`
- **Support accents:** `#7f9186` (sage), `#7f8795` (slate), `#b29f7f` (sand)

## Installation (local theme folder)

If this folder is your theme source, copy or symlink files into your Omarchy config/theme location,
then apply it with:

```bash
omarchy-theme-set "Urban Horizon"
```

## Theme features

- Light, low-stress UI with medium contrast
- Soft rounded corners and subtle transparency
- Muted accent borders and neutral module colors
- Consistent palette across Hyprland, Waybar, terminals, GTK, editor themes, and utilities
