# Aurora

**A cosmic wilderness theme for Omarchy** — Inspired by the aurora borealis dancing over camping landscapes, blending deep night skies with vibrant **campfire orange** accents and warm wilderness tones.

## Color Palette

- **Background:** Deep Night Blue (`#1a1f2e`)
- **Foreground:** Soft Cloud White (`#e5e9f0`)
- **Primary Accent:** Campfire Orange (`#d08770`) - Warm ember glow
- **Secondary Accents:**
  - **Bright Orange:** `#d98770` - Intense flame
  - **Purple:** Twilight Sky (`#b48ead`) - Soft purple from sunset
  - **Green:** Forest Moss (`#a3be8c`) - Deep forest green
  - **Blue:** Deep Sky (`#5e81ac`) - Rich cosmic blue
  - **Yellow:** Sunset Gold (`#ebcb8b`) - Golden hour warmth

## Installation

Paste the command in your terminal:

```bash
omarchy-theme-install https://github.com/TimBLukas/aurora-nights.git
```

**OR**

- Open Omarchy Menu (Super + Alt + Space)
- Go to Install > Style > Theme and paste the git repo link:

```bash
https://github.com/TimBLukas/aurora-nights.git
```

## Theme Features

### Visual Style

- **Subtle Transparency:** More translucent windows for an ethereal, layered look
- **⭕ Soft Rounded Corners:** 10px border radius for a cozy, modern feel
- **Atmospheric Shadows:** Enabled with aurora-teal glow
- **Smooth Animations:** Slow, dreamy transitions matching the cosmic wilderness aesthetic
- **Medium Spacing:** Balanced 5-6px gaps between windows

### Opacity Settings

- Active windows: 88% (more transparent)
- Inactive windows: 82% (even more transparent)
- Fullscreen: 100% (fully opaque)

## Custom Waybar Features

### Layout

- **Position:** Top bar
- **Height:** 30px (slim, compact)
- **Style:** Modern centered design with info on sides
- **Rounding:** 10px (matches window style)

### Modules Included

**Left Side:**

- Custom launcher button (opens Walker)
- Workspace indicators with numbers
- Window title with app icons

**Center:**

- Clock with date toggle (24-hour format)
- Interactive calendar tooltip

**Right Side:**

- CPU usage
- RAM usage
- Disk usage
- Temperature monitor
- Volume control
- Network status
- Battery indicator
- System tray
- Power menu button

### Visual Features

- **Orange glow on hover** - Matches campfire accent color
- **Active workspace highlighted** - Orange border and background
- **Icon-only display** - Clean, minimal aesthetic
- **Color-coded states:**
  - CPU - Blue
  - Memory - Green
  - Disk - Purple
  - Network - Yellow
  - Warnings - Orange
- **Smooth animations** - All interactions have 0.3s transitions
- **Pulsing alerts** - Critical states animate for attention

## Design Philosophy

Aurora blends the mystical beauty of wilderness camping with warm campfire aesthetics. The theme uses:

- **Deep blues** from night skies as the foundation
- **Campfire orange** as the primary accent (warm, inviting glow)
- **Warm tones** for highlights and interactive elements
- **Soft purples** from twilight skies
- **Forest greens** for natural balance

The result is a cosmic yet cozy environment perfect for late-night coding sessions by the campfire.
