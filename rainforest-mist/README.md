# Misty Rainforest

**A serene foggy wilderness theme for Omarchy** — Inspired by misty mornings in dense rainforests, where soft fog drifts through ancient trees and moss-covered stones. Cool, dreamy, and ethereal with **soft moss green** accents and water-kissed turquoise tones.

## Color Palette

- **Background:** Deep Forest Shadow (`#181d1a`)
- **Foreground:** Misty Cloud (`#d8e5dd`)
- **Primary Accent:** Soft Moss (`#7a9e7e`) - Muted foggy green
- **Secondary Accents:**
  - **Turquoise:** `#6ba89f` - Mountain stream water
  - **Jade:** `#5a8e7a` - Deep rainforest green
  - **Fog Gray:** `#7a8c8e` - Cool mist
  - **Fern:** `#8eb891` - Bright forest vegetation
  - **Stone:** `#5e6b6a` - Moss-covered rocks

## Installation

Paste the command in your terminal:

```bash
omarchy-theme-install https://github.com/TimBLukas/misty-rainforest.git
```

**OR**

- Open Omarchy Menu (Super + Alt + Space)
- Go to Install > Style > Theme and paste the git repo link:

```bash
https://github.com/TimBLukas/misty-rainforest.git
```

## Theme Features

### Visual Style

- **Heavy Fog Transparency:** Very translucent windows (75-80%) for a dreamy, fog-drenched atmosphere
- **⭕ Soft Rounded Corners:** 10px border radius for organic, natural feel
- **Diffused Shadows:** Enabled with soft moss-green glow
- **Slow Drift Animations:** Gentle, floating transitions like mist through trees
- **Medium Spacing:** Balanced 5-6px gaps between windows

### Opacity Settings

- Active windows: 80% (heavy fog)
- Inactive windows: 75% (very dreamy, ethereal)
- Fullscreen: 100% (fully opaque)

## Custom Waybar Features

### Layout

- **Position:** Top bar
- **Height:** 30px (slim, compact)
- **Style:** Modern centered design with info on sides
- **Rounding:** 10px (matches window style)

### Modules Included

**Left Side:**

- Custom launcher button (opens Walker)
- Workspace indicators with numbers
- Window title with app icons

**Center:**

- Clock with date toggle (24-hour format)
- Interactive calendar tooltip

**Right Side:**

- CPU usage
- RAM usage
- Disk usage
- Temperature monitor
- Volume control
- Network status
- Battery indicator
- System tray
- Power menu button

### Visual Features

- **Moss-green glow on hover** - Matches soft rainforest accent

- **Smooth animations** - All interactions have 0.3s transitions
- **Pulsing alerts** - Critical states animate for attention

## Design Philosophy

Misty Rainforest captures the serene, ethereal beauty of fog-drenched ancient forests. The theme uses:

- **Deep forest shadows** as the foundation
- **Soft moss green** as the primary accent (muted, organic)
- **Cool tones** for a misty, dreamy atmosphere
- **Blue-gray fog** for depth and mystery
- **Turquoise water** accents for vibrancy

The result is a calm, ethereal environment perfect for focused work in a peaceful rainforest sanctuary.
